Travel Power

Travel Power Adapters by Country

Which plug and voltage each destination uses, whether you need an adapter or a converter, and what to pack. Pick a country to get the details.

Most travel electronics are dual-voltage, so for the majority of trips you need only the right plug adapter — not a heavy voltage converter. These guides tell you exactly which is which, country by country.

East Asia

  • China — Type A, C and I, 220 V
  • Japan — Type A and B, 100 V
  • South Korea — Type C and F, 220 V

Southeast Asia

  • Singapore — Type G, 230 V
  • Thailand — Type A, B, C and O, 220 V
  • Vietnam — Type A, C and F, 220 V

South Asia

  • India — Type C, D and M, 230 V

Europe

  • France — Type C and E, 230 V
  • Germany — Type C and F, 230 V
  • Italy — Type C, F and L, 230 V
  • Spain — Type C and F, 230 V
  • Switzerland — Type C and J, 230 V
  • United Kingdom — Type G, 230 V

North America

  • Canada — Type A and B, 120 V
  • United States — Type A and B, 120 V

Oceania

  • Australia — Type I, 230 V
  • New Zealand — Type I, 230 V

Middle East

  • United Arab Emirates — Type C, D and G, 230 V

South America

  • Brazil — Type C and N, 127 / 220 V

Africa

  • South Africa — Type D, M and N, 230 V
Travel Now tip: One good universal adapter with USB-C covers almost everywhere in this list. See any country page for a current pick.
