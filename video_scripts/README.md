# video_scripts/

Short-form video scripts generated from Travel Now Markdown drafts.

Each run creates a folder: `YYYY-MM-DD-{slug}/`

---

## What's in each folder

| File | Contents |
|---|---|
| `scripts.md` | YouTube Shorts, Instagram Reels, and TikTok scripts — hook, full script, visual cue |
| `production_notes.md` | Voiceover script (platform-neutral), on-screen text overlay order, shot list |
| `posting_kit.md` | Social media caption, hashtags, CTA text |
| `video_gen_prompt.md` | Detailed prompt for a future AI video generation tool (reference only) |

---

## Generating scripts

```bash
# List available input drafts
python generate_video_scripts.py --list-drafts

# Dry run (no API call, no files written)
python generate_video_scripts.py \
  --input content_drafts/2026-05-29-airport-security-liquids.md \
  --mode travel_now

# Generate and save
python generate_video_scripts.py \
  --input content_drafts/2026-05-29-airport-security-liquids.md \
  --mode travel_now --write

# Japanese AI media series
python generate_video_scripts.py \
  --input note_drafts/2026-05-29-paid-ai-travel-media-7days-workflow-practical.md \
  --mode japanese_ai_media --write

# Regenerate (overwrite existing folder)
python generate_video_scripts.py --input ... --mode ... --write --force
```

---

## Modes

| Mode | Language | Voice | CTA |
|---|---|---|---|
| `travel_now` | English | Practical, helpful, non-spammy | Travel Now Checklist Generator |
| `japanese_ai_media` | Japanese | Poor young worker, light Kansai-ben, honest | note series (free + paid articles) |

---

## Production workflow (manual)

```
Step 1: Review scripts.md — choose the best hook for each platform
Step 2: Review production_notes.md — use voiceover script + shot list to film
Step 3: Record and edit the video
Step 4: Use posting_kit.md — copy caption + hashtags + CTA when posting
Step 5: Post manually to YouTube Shorts / Instagram Reels / TikTok
Step 6: Save video_gen_prompt.md for future use with AI video generation tools
```

---

## Important constraints

- Scripts are generated by Gemini AI. Always review before recording.
- Do not post automatically. Human judgement required.
- Do not use YouTube, TikTok, Instagram, or Meta APIs.
- No video generation APIs are used — scripts are text only.
- Travel Now CTAs point to: https://kytriples.github.io/travel-now-agent/checklist-generator.html
- Japanese AI media CTAs point to the note series (update CTA with actual note URLs after posting).
- No exaggerated income claims. No guaranteed results. No "secret method" language.
- Soft wording required for safety/legal/medical topics.
