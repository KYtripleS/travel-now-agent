**① APIキー漏洩の注意喚起**
```
AIを使う副業を始めるとき、最初にやること。

.envファイルにAPIキーを書く。
.gitignoreに.envを追加する。

これだけで「うっかりGitHubに公開してしまう」事故を防げます。
地味だけど、絶対に最初にやっておくことの一つです。

#AI副業 #初心者向け
```

**② GitHub Pages紹介**
```
サイトを持ちたいけどサーバー代が気になる人へ。

GitHub Pagesを使えば、HTMLファイルを置くだけで
https://ユーザー名.github.io/〜 として無料で公開できます。

コード初心者でも、AIに手順を教えてもらいながら作れます。
まず「公開できた状態」を作ることが全ての始まりです。

#GitHub #副業 #AI副業
```

**③ 最初はこれだけでいい**
```
AI副業メディアを始めるなら、最初に必要なファイルはこれだけ。

・index.html（サイトの入り口）
・.env（APIキーを書く）
・.gitignore（.envをGitに含めない）
・README.md（自分向けのメモ）

スクリプトや自動化は後から追加できます。
まず「サイトが存在する状態」を作ることが先です。

#AI副業 #初心者
```
