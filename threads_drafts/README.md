# threads_drafts/

Threads post drafts generated by `generate_threads_posts.py`.

## Rules

- All drafts are for **manual posting only**.
- Do NOT post automatically.
- Do NOT use the Meta API or Threads API.
- Do NOT log in via script.
- Do NOT use browser automation.

## File naming

```
YYYY-MM-DD-japanese_ai_media.md
YYYY-MM-DD-travel_now.md
```

## Usage

```bash
python generate_threads_posts.py                         # dry run
python generate_threads_posts.py --write                 # japanese_ai_media (default)
python generate_threads_posts.py --write --mode travel_now
python generate_threads_posts.py --write --all           # both modes
python generate_threads_posts.py --write --force         # overwrite today's files
```

## Each draft contains

- **5 standalone posts** — self-contained, no CTA or soft CTA
- **2 thread sequences** — 2–3 connected posts on one topic
- **3 soft CTA variants** — posts that invite readers to check a note article or the checklist generator
