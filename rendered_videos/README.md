# rendered_videos/

Local MP4 files generated by `make_video.py` for manual upload.

**Never posted automatically. Human review and upload required.**

---

## Generating a video

```bash
# Install dependencies (once)
pip install Pillow imageio imageio-ffmpeg

# Generate the Before You Fly video
python make_video.py

# Preview a single slide as PNG (fast, no MP4 written)
python make_video.py --preview-frame 1   # hook
python make_video.py --preview-frame 2   # check 1
python make_video.py --preview-frame 9   # end card

# Custom output path
python make_video.py --output rendered_videos/my-cut.mp4

# Skip cross-dissolves (faster render)
python make_video.py --no-transitions
```

## Posting workflow (manual only)

```
Step 1: open rendered_videos/YYYY-MM-DD-before-you-fly.mp4
        → open rendered_videos/2026-05-30-before-you-fly.mp4

Step 2: review all slides play correctly

Step 3: (optional) add music + captions in CapCut / DaVinci Resolve / YouTube Studio

Step 4: upload manually:
  YouTube Shorts  → youtube.com/upload → mark as Short
  Instagram Reels → instagram.com → + → Reel → select MP4
  TikTok          → tiktok.com/upload → select MP4

Step 5: paste caption + hashtags from:
  video_scripts/2026-05-30-before-you-fly-5-things-to-check/posting_kit.md

Step 6: update link in bio with:
  https://kytriples.github.io/travel-now-agent/checklist-generator.html
```
